  • Publication Statistics
     Description
This view offers a summary of the total number and volume of products published in the Data Hubs for each Copernicus Sentinel mission, within the selected time interval: last 24 hours, last 7 days, last 30 days and last 3 months (i.e. 90 days) with respect to the current date / time. The view provides also the details about the published products per product level or sensor type (i.e., for Copernicus Sentinel-3). Charts are interactive: by clicking on a label in the legend it is possible to modify the displayed results.
